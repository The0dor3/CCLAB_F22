Translate this!

How to play

  1. Click on the image
  2. Type the translation of the Chinese word into the text box: Only type lowercase; spaces are needed for certain words.
  3. Reload the page for a different background
  4. Have fun!
(p.s. I have added a pinyin list so that you can learn the pronunciations)

Disclaimer:

In this game there is only one English translation for each Chinese word. In reality, however, there are many possible translations of each word. If you get a translation wrong that should have the same meaning, remember what the answer for this game is, because these answers will not always be the only right answer :)
